# Bem-vind@ ao repositório do desafio "Make Magic" da Dextra

Para sua conveniência, um ambiente de desenvolvimento Vagrant está disponível na pasta `vagrant`, e pode ser inicializado utilizando o comando `vagrant up` dentro da mesma pasta. Toda inicialização será feita, incluindo o download de qualquer dependência, e sua aplicação estará disponível no endereço [192.168.9.34](http://192.168.9.34).
